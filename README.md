# Financial-Management-Dashboard
Dashboard interativo desenvolvido em Power BI para análise de fluxo de caixa e saúde financeira empresarial. O projeto foca em transformar dados brutos de extrato em insights acionáveis através de uma interface moderna e intuitiva.
![Financial-Management-Dashboard](FinancialManagementDashboardimage,png)

- Diferenciais do Projeto
Interface UI/UX de Alto Impacto: Design baseado em Glassmorphism com menu lateral de navegação e paleta de cores otimizada para legibilidade.

Lógica DAX Avançada: Implementação de medidas calculadas para segregação de fluxos (Receitas vs. Despesas) e cálculo de saldo líquido em tempo real.

Visual Data Storytelling: Gráficos integrados que permitem analisar a evolução mensal e a composição de custos por categoria de forma simultânea.

-  Tecnologias Utilizadas
Power BI Desktop (Modelagem e Visualização).

DAX (Cálculos de inteligência de tempo e filtros de contexto).

ETL / Power Query (Tratamento e padronização dos dados).

-  KPIs Principais
Receita Total: Monitoramento de entradas brutas.

Despesas Totais: Controle granular de saídas por unidade e categoria.

Saldo Líquido: Cálculo automático de margem operacional.
